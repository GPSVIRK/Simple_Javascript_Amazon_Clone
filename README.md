# Simple JavaScript Amazon Clone

A simple front-end Amazon clone built using **HTML, CSS, and JavaScript**.  
This project was created by following the tutorial by **SuperSimpleDev** and recreates core e-commerce functionality such as product listing, shopping cart behavior, and a checkout page.

Repository: https://github.com/GPSVIRK/Simple_Javascript_Amazon_Clone

## Features

- Product catalog display
- Add to cart functionality
- Cart quantity updates
- Order summary page
- Basic checkout interface
- Modular JavaScript code structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
